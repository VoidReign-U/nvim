## Packages Needed To Install To Work Perfectly

### Arch Linux

```sh
sudo pacman -S neovim go python-pip ripgrep fd curl wget git unzip nodejs npm
```
### Debian && Ubuntu

```sh
sudo apt install neovim python3-pip ripgrep fd-find curl wget git unzip nodejs npm
```
### Fedora

```sh
sudo dnf install neovim python3-pip ripgrep fd curl wget git unzip nodejs npm
```
### Void Linux

```sh
sudo xbps-install -S neovim python3-pip ripgrep fd curl wget git unzip nodejs npm
```
